  <action>
Create the utility module and four SVG sub-components. All are pure rendering components with no state or side effects.

**gantt-utils.ts:**
- Export constants: `DAY_WIDTH = 34`, `ROW_HEIGHT = 34`, `BAR_HEIGHT = 22`, `BAR_Y_OFFSET = (ROW_HEIGHT - BAR_HEIGHT) / 2` (= 6), `HEADER_HEIGHT = 40` (two 20px rows for month + day)
- Export `DateRange` interface: `{ startDate: Date; endDate: Date; totalDays: number }`
- Export `computeDateRange(tasks: ComputedTask[], paddingBefore = 7, paddingAfter = 14): DateRange` -- finds min effectiveStartDate and max endDate across ALL tasks (not just visible), adds padding. Returns empty-safe fallback (today +/- padding) for zero tasks. Uses `date-fns` parseISO, min, max, subDays, addDays, differenceInCalendarDays.
- Export `dateToX(dateISO: string, rangeStart: Date): number` -- returns `differenceInCalendarDays(parseISO(dateISO), rangeStart) * DAY_WIDTH`

**GanttBar.tsx:**
- Props: `{ task: ComputedTask; rowIndex: number; rangeStart: Date; ownerColor: string }`
- Renders an SVG `<g>` containing:
  1. Background `<rect>` at `(x, y, width, BAR_HEIGHT)` with `fill={ownerColor}` at `opacity={0.35}`, `rx={3}` for rounded corners
  2. Completion fill `<rect>` (same position, width = `barWidth * (task.completionPct / 100)`) at `opacity={0.85}` -- only render if completionPct > 0
- x = `dateToX(task.effectiveStartDate, rangeStart)`, width = `dateToX(task.endDate, rangeStart) - x`, minimum width = `DAY_WIDTH` (one day)
- y = `HEADER_HEIGHT + rowIndex * ROW_HEIGHT + BAR_Y_OFFSET`
- Default color constant: `#94A3B8` (slate-400) when no ownerColor provided

**GanttTimescale.tsx:**
- Props: `{ range: DateRange }`
- Uses `date-fns` eachDayOfInterval to iterate days in range
- Top row (y=0 to y=20): month+year groups. Group consecutive days by `getMonth()+getYear()`, render a rect + text label (e.g., "Mar 2026") for each month span.
- Bottom row (y=20 to y=40): day number cells. Each cell is `DAY_WIDTH` wide. Weekend days (`isWeekend()`) get a subtle background fill using `var(--muted)`.
- Use `stroke="var(--border)" strokeWidth={0.5}` for cell borders.
- Text fills use `var(--muted-foreground)`.

**GanttDependencyArrow.tsx:**
- Props: `{ sourceEndX: number; sourceY: number; targetStartX: number; targetY: number }`
- Renders an SVG `<path>` with right-angle connector: horizontal out from source center-Y, vertical to target center-Y, horizontal in to target.
- Path: `M sourceEndX sourceCenterY H midX V targetCenterY H targetStartX` where midX = average of sourceEndX and targetStartX, centerY = y + BAR_HEIGHT/2
- Stroke: `#94A3B8` (muted gray), strokeWidth 1.5, fill none
- Add a small arrowhead at the target end (4px triangle pointing right)

**GanttTodayMarker.tsx:**
- Props: `{ rangeStart: Date; totalHeight: number }`
- Computes today's X position via `dateToX(format(new Date(), 'yyyy-MM-dd'), rangeStart)`
- Renders a vertical `<line>` from y=0 to y=totalHeight in `#EF4444` (red-500), strokeWidth 1.5, with `strokeDasharray="4 3"` for a dashed line
- Only renders if today falls within the visible range (x >= 0 and x <= totalDays * DAY_WIDTH)

  <action>
Create the main GanttChart component that composes all sub-components into a single SVG element.

**GanttChart.tsx:**
- Props interface `GanttChartProps`:
  - `visibleTasks: ComputedTask[]` -- the filtered, tree-sorted tasks (same array the table renders)
  - `allTasks: ComputedTask[]` -- all tasks for date range calculation (not filtered by collapse)
  - `owners: Owner[]`
  - `dependencies: Dependency[]`

- Build `ownerMap: Map<string, Owner>` from owners array for O(1) color lookups.

- Compute `range = computeDateRange(allTasks)` -- uses ALL tasks so date range doesn't jump on collapse.

- Compute `svgWidth = range.totalDays * DAY_WIDTH`
- Compute `totalHeight = HEADER_HEIGHT + visibleTasks.length * ROW_HEIGHT`

- Build a `taskIndexMap: Map<string, number>` mapping each visible task ID to its row index for dependency arrow positioning.

- Render weekend column shading: iterate `eachDayOfInterval(range)`, for each weekend day render a full-height `<rect>` at `(dayIndex * DAY_WIDTH, HEADER_HEIGHT, DAY_WIDTH, visibleTasks.length * ROW_HEIGHT)` with `fill="var(--muted)" opacity={0.3}`.

- Render the SVG:
  ```
  <svg width={svgWidth} height={totalHeight} style={{ flexShrink: 0, display: 'block' }}>
    <GanttTimescale range={range} />
    {/* Weekend shading rects */}
    {/* Bars */}
    {visibleTasks.map((task, idx) => (
      <GanttBar key={task.id} task={task} rowIndex={idx} rangeStart={range.startDate}
        ownerColor={task.ownerId ? ownerMap.get(task.ownerId)?.color ?? '#94A3B8' : '#94A3B8'} />
    ))}
    {/* Dependency arrows -- only for pairs where BOTH tasks are visible */}
    {dependencies.filter(dep =>
      taskIndexMap.has(dep.upstreamTaskId) && taskIndexMap.has(dep.downstreamTaskId)
    ).map(dep => {
      const srcTask = visibleTasks[taskIndexMap.get(dep.upstreamTaskId)!];
      const tgtTask = visibleTasks[taskIndexMap.get(dep.downstreamTaskId)!];
      const srcIdx = taskIndexMap.get(dep.upstreamTaskId)!;
      const tgtIdx = taskIndexMap.get(dep.downstreamTaskId)!;
      return <GanttDependencyArrow
        key={dep.id}
        sourceEndX={dateToX(srcTask.endDate, range.startDate)}
        sourceY={HEADER_HEIGHT + srcIdx * ROW_HEIGHT + BAR_Y_OFFSET}
        targetStartX={dateToX(tgtTask.effectiveStartDate, range.startDate)}
        targetY={HEADER_HEIGHT + tgtIdx * ROW_HEIGHT + BAR_Y_OFFSET}
      />;
    })}
    <GanttTodayMarker rangeStart={range.startDate} totalHeight={totalHeight} />
  </svg>
  ```

- Wrap SVG computation in useMemo for range, ownerMap, taskIndexMap.
- This component does NOT handle scroll or layout -- that is ScheduleClient's job in Plan 02.
